WiFi-Driven MoBox for Real-Time Analysis

An innovative mobile drop box system designed to track and manage device occupancy and attendance using Wi-Fi-based detection technology. It uses the ESP8266 Wi-Fi module to detect and count devices like smartphones and tablets inside a confined area by identifying their unique MAC addresses.

A dedicated web interface allows different levels of access for admins, HODs, and tutors to manage attendance remotely. An aluminum-lined box blocks external signals for accurate detection.

  • Technologies: ESP8266, Wi-Fi, HTML/CSS, Arduino IDE
  • Use Cases: Educational institutions, libraries, labs
  • Outcome: Published in IJSART for smart monitoring innovation
